# ComfyUI Workflows

This folder contains ready-to-use ComfyUI workflows for the Private AI Stack.

## Available Workflows

### Juggernaut XL v9 (SDXL)

- File: `juggernaut-xl-v9.json`
- Base model: Juggernaut XL v9 (SDXL)
- Supports:
  - LoRA loading
  - High-res generation
  - Creative & cinematic styles

### How to Import

1. Open ComfyUI
2. Click “Load” or drag the JSON file into the canvas
3. Ensure required models exist under:
`/mnt/ai/comfyui/root/models/`
4. Adjust prompt, resolution, and generate

## Notes

- Workflows assume:
- GPU available
- SDXL-compatible setup
- Juggernaut XL v9 checkpoint installed
